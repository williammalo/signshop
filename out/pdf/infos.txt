 All template files are updated with the file names to show
 what year the model is, example: FORD-EXPLORER-2003-UPTODATE.EPS
 will mean that this model in 2003 is the same in 2010, however
 when you import the layout it might read FORD-EXPLORER-2003
 only, but it will be the 2010 models.

-ABBREVIATIONS/LEGEND:
Here is the definition on the file name abbreviations:
DR = DOOR
4DR = 4 SIDE DOORS
5DR = 5 SIDE DOORS
CREW-C or C-C= CREW CAB SIDE DOORS
EXT-C = EXTENDED CAB
REG = REGULAR VERSION
CARGO-S = CARGO SWING SIDE DOOR
SLIDE or S = SLIDE SIDE DOOR
EXT = EXTENDED VERSION
Q = QUAD CAB (RAM PICKUPS)
GR = GRANDE VERSION
W = WAGON VESION
5-5FT = BOX OF PICKUP TRUCK OF 5 FEET 5 INCHES
-F- = FLAIRSIDE STYLE PICKUP BOX (OR DOOLY STYLE WHEEL BODY)
HD = HEAVY DUTY
SP = SPORT VERSION
HB = HATCH BACK VESION
CON = CONVERTIBLE VERSION
